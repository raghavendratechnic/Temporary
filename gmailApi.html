Welcome to Gmail API for sending emails
with your own email address!
How to use

Just send a POST request to https://gmailapi.vercel.app with a body that contains these parameters: sender, password, recipient, subject, and message.

Parameters and description

1. sender = your Gmail address
2. password = Gmail app password learn more
3. recipient = receiver email address
4. subject = email subject
5. message = message (HTML allowed)

Example in BJS
HTTP.post({ url: "https://gmailapi.vercel.app", success: "onsuccess", body: { sender: "hello@gmail.com", password: "dfezbchlnvryacfi", recipient: "hii@gmail.com", subject: "Subject here", message: "Simple message here (HTML allowed)." } })
